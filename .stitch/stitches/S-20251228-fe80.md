## Intent

Add `--json` (`-j`) flag to `stitch list` and `stitch status` CLI commands to enable machine-readable output for scripting.

## Changes

- Added `renderStitchListJson()` function to `src/core/render.ts`
- Added `renderStatusJson()` function to `src/core/render.ts`
- Updated `stitch list` command to accept `-j, --json` flag
- Updated `stitch status` command to accept `-j, --json` flag

## Usage

```bash
stitch list --json          # JSON array of all stitches
stitch list -s open --json  # Filter + JSON output
stitch status --json        # Current stitch and lineage as JSON
```
